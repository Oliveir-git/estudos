INTRODUÇÃO CSS

Ao falarmos de css, estamos dizendo sobre Cascading styles sheets (Folhas de estilos em cascatas) Ou seja, todo estilo que aplicarmos a CSS sera redirecionado ao referido HTML como por exemplo diz que o html selecionado ira mudar de cor para "preto" ou até mesmo o tipo de fonte como font-family="sanserif";que mudaria a fonte para sem serifa
Feito especialmente para suprir a necessidade da tag <font> onde os desenvolvedores ficam perdidos em edições em sites maiores. Com a CSS podemos separar arquivos especificios com formatos pre selecionados para que o HTML se adapte. Como por exemplo esta formatação onde demos estilo ao p (paragrafo)
<style>
p {
color: white;
background-color: black;
font-family: "Arial, Consolas, sanserif"
}
</style>

Selector Aqui é onde colocamos qual tag iremos dar estilo, como no exemplo acima, estamos selecionando o h1 para darmos inicio ao CSS.
Declaration Uma declaração é feita com dois elementos, a property(Proriedade) onde colocamos o que iremos mudar no elemento, como a color, backgorund-image (imagem de fundo) ou a font-family (estilo da letra ) e em seguida temos o value (valor) onde adicionamos qual cor, ou qual o tamanho, ou qual o tipo de letra que iremos mudar.

CSS Selector seletores

Seletores de CSS pode indicar qual elemento do html queremos estilizar... Podemos citar alguns exemplos.

Element Selector

Seletores de Elementos, podemos indicar o elemento (h1, p, div, span, body, hmtl etc...) sao uteis pois assim como os demais, podemos indicar mais de um seletor chamados de Group Selecetors como por exemplo.
<style>
h1, h2, p, body {
font-family = "Arial, sanserif"
}
</style>
Isto, muda os elementos h1,h2,p,body deixando todos com o estilo Arial como fonte.

ID and Class Selector

Muito uteis e mais utilizadas, pois com id's (#) e classes (.) permite ao css que estilize, somente os elementos que contenham a classe ou id criados.
Podemos criar estes curiosos amigos de varias formas, e a good pratice é sempre nomea-los de acordo com sua função como por exemplo
<style>
.cor {
color:white; backgorund-color:"black"
}
</style>
O segredo é sempre atribuilos de acordo com suas funções para que o codigo fique limpo e legivei
a diferença entre Id's e classes, é que ID nao pode utilizar em mais de um elemento, ficaria exclusivo para um unico uso, enquanto class sao utilizados sem limite de uso em elementos
Ou seja, Ids sao utilizados para uma formatação espeficica em um unico elemento, como o main ou o proprio html
(OBS = CLASSES NAO PODEM INICIAR COM NUMEROS EM SEUS NOMES)

Universal Selector (*)

Indicado apenas por * mas uma poderosa ferramenta onde as formatações nele sao aplicadas para todos os elementos
por exemplo.
<style>
* {
padding: 0 ; margin: 0;
}
</style>
Mudaria todo elemento, independente de qual seja p,html,body,div,span todos vao ter padding 0px e margin 0px.

COLORS

Cores são exenciais no estilo de seu website
Podemos estilizar as cores com um simples comando. <style>
h1 {color : black;}
</style>
Com a adiçao do atributo mais simples color fazemos com que a cor mude para o formato que quisermos, seja RGB HSB ou em hexadecimal.

BACKGROUNDSS

  • backgorund-color Funciona da mesma forma que estilizar um texto, sempre indicando um elemento (h1, p, div, span ... etc) onde funcioanara como uma box, e nela damos o seguinte codigo h1 { background-color: blue}

  • background-image Outra forma de estilizar o fundo do site, é com o parametro div {background-image: url(destino)}; onde podemos adicionar qualquer imagem que tenhamos como background, altamente recomendavel
  • background-repeat Por padrão, uma imagem que ultrapassa a resolução da imagem com sua medida se repete até preencher todo o fundo sem que dimnua sua qualidade.
    Podemos mudar isto, e ainda alterar alguns atributos.
    background-repeat: no-repeat - retira totalmente a repetição da imagem, normalmente temos que corrigir a dimensao da imagem por conta disto
    background-repeat: repeat-y (horizontal)-
    background-repeat: repeat-x (vertical)
  • background-attachment Ligaçao de background - especifica se o fundo ira ficar fixed (colado no fundo da pagina) ou scroll rolando junto conforme abaixamos a roda do mouse
  • background-position Definindo a centrialização da imagem onde podemos colocar os valores = top, bottom, left, onde pode ser inserido ate 2 valores, dizendo por exemplo se o fundo fica h1 {background-position: center center} ou uma imagem centrializada para baixo e a esquerda do layout: h1 {background-position: left bottom}
