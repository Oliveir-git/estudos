Meta Tags (Tags de cabeçalhos)

Description MUITOO IMPORTANTE

A metatag Description e invisivel ao usuario, mas é usada para auxiliar os buscadores do google com a descriçção de sua pagina ... <meta name="description" content="Este é a descrição da pagina como por exemplo = São milhares de Cursos, DevCasts, Códigos e Artigos sobre Java, JavaScript, PHP, Delphi, Banco de dados, HTML, CSS, C#, C++, Python e .NET. Acesse agora!"
Conseguimos colocar varias com o que podemos chamar de hashtags que vao auxliar o buscador.

Robots

Outra Tag muitoo IMPORTANTE

com o uso da tag <meta name="robots" content="noindex"> Faz com que o navegador reconheça que aquela pagina nao sera incluida nos buscadores do google, util para paginas de formularios, ou administrativas onde impeça a entrada direta do usuario.

Nofolow

Outro util que nao se é usado muito, é pra quando queremos que separe a associação de links da nossa pagina como links que usuarios enviam em comentarios por exemplo.
Assim os buscadores nao assemelham links 3ceiros (OBS podemos utilizar nofolow e noindex desta forma : <meta name="robots" content="noindex, nofolow")

Como utilizar tamanhos proporcionais para nao danificar e destorcer a qualidade ao da imagem ao utilizar width e height
Uma imagem em estilo retrato pode ter uma proporção de 2:3. Nesse caso, a altura é 1,5(resto) vez maior do que a largura. Portanto, a imagem pode ter 500 px × 750 px, 1.500 px × 2.250 px etc. imagens 1:1 são imagens de tamanho iguais (quadradas)

Lista de Definição

  • <dl> </dl> - abertura e fechamento de uma lista;
  • <dt> </dt> - abertura e fechamento de um termo;
  • <dd> </dd> - abertura e fechamento de uma descrição.
    •

Exemplo de como se utilizar uma lista de denição com a semantica correta

<dl>
<dt>Front-End</dt>
<dd>Toda parte visual de um site, onde o usuário interage diretamente.</dd>
<dt>Back-End</dt>
<dd>Estrutura por trás das aplicações, que suporta todas as ações do usuário.</dd>
</dl>

Exemplos de como utilizar <ol type=""

<ol type="1" - Ordenação numérica (padrão)
<ol type="a" - Ordenação alfabetica minúscula
<ol type="A" - Ordenação alfabetica maiúscula
<ol type="i" - Numerais romanos minúsculos
<ol type="I" - Numerais romanos maiúsculos
ou
<ul type="disc" Faz uma bolinha cheia
<ul type="square" Faz uma lista de quadrados
<ul type="circle" - Faz uma lista com bolinhas
<ul type="none" - Remove a marcação da lista

UM JEITO INTERESSANTE DE USAR LISTAS, É QUE PODEMOS FAZER LISTAS DENTRO DE LISTAS

Exemplo

X
<ul> //Ordenação principal
<li> //Lista principal
<h3>//TITULO DA LISTA </h3>
